🐯 API Server for Code Archives 🐯

【使用技術】  
- クライアントサイド
  - 言語： JavaScript(React 16.13.1)
  - フレームワーク: Next.js 10.0.3  
  - デプロイ： Vercel
  - コードハイライト: Prism.js  
  
- サーバーサイド
  - 言語： go 1.15.2  
  - データベース: CloudSQL(MySQL 8.0)  
  - デプロイ： Google App Engine

【機能一覧】  
- コードの新規登録・編集・削除  
- 検索機能  
- ユーザー登録・認証機能  

アプリのurl: https://code-archives-client.vercel.app/
